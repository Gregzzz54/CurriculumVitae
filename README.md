# Curriculum Vitae

Personal CV as a static GitHub Pages site.

GitHub profile: [Gregzzz54](https://github.com/Gregzzz54/)

- [`index.html`](index.html) — full CV layout and content, served directly by GitHub Pages.
- [`Convert-CV.ps1`](Convert-CV.ps1) — converts `index.html` to `.docx` and `.pdf` using Microsoft Word.

## Export to Word and PDF

Requires Microsoft Word to be installed.

```powershell
.\Convert-CV.ps1
```

Outputs `Nathan-Gregory-CV.docx` and `Nathan-Gregory-CV.pdf` in the same directory.
